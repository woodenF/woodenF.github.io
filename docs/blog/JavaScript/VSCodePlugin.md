# VSCode 插件
> 部分实用的VSCode插件

* ```Bracket Pair Colorizer``` 着色匹配括号
* ```Indent-Rainbow``` 着色显示文本前方的缩进
* ```Trailing Spaces``` 检测多余空格并高亮
* ```TODO Highlight``` TODO备忘插件
* ```HTML CSS Support``` 智能提示
* ```GitLens``` 增强了VSCode中内置的git功能
* ```SVG Viewer``` 在VSCode内查看svg图标
